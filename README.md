## Installation
Necessary to have root rights
```bash
git clone https://github.com/hacknonym/GenPhish.git
cd GenPhish
chmod +x genphish.sh
./genphish.sh --setup
```
### Usage 
```bash
cd GenPhish
./genphish.sh --help
```
### Update 
```bash
cd GenPhish
./genphish.sh --update
```
